## 📱 Guide interactif

Le fichier `roadtrip_guide.jsx` est un composant React interactif avec :
- Navigation jour par jour
- Boutons **Waze**, **Google Maps** et **Apple Maps** pour chaque destination
- Checklist interactive avant le départ
- Tous les liens utiles (réservations, météo, sites officiels)

## 📄 PDF

Le fichier `roadtrip_suisse.pdf` contient le guide complet imprimable (9 pages).
